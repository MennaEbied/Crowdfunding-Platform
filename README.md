# Crowdfunding Platform
A full-stack web application built to facilitate community-driven project funding


## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** JSON Server (REST API)
- **Security:** Custom Middleware for Role-Based Access Control
  

## ✨ Core Features
- **Multi-Role Authentication:** Distinct interfaces and permissions for Guests, Registered Users, and Admins.
- **Campaign Lifecycle:** Users can create, edit, and track their funding projects.
- **Admin Suite:** Full moderation control to approve/reject campaigns and manage user account status.
- **Dynamic UI:** A responsive navbar and dashboard that adapts based on the logged-in user's role
